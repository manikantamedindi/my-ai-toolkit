# OpenCode AI Toolkit

A collection of reusable AI Skills, Agents, Prompt Templates, and Workflows for OpenCode.

## Repository Structure

```
my-ai-toolkit/
├── README.md
├── LICENSE
├── CHANGELOG.md
├── skills/          — Reusable AI skills by domain
│   ├── frontend/
│   ├── react/
│   ├── vue/
│   ├── nextjs/
│   ├── nuxt/
│   ├── javascript/
│   ├── typescript/
│   ├── nodejs/
│   ├── ui-ux/
│   ├── accessibility/
│   ├── performance/
│   ├── seo/
│   ├── testing/
│   └── git/
├── agents/          — Specialized coding agent definitions
├── prompts/         — Reusable prompt templates
├── templates/       — Project templates (nextjs, vue, react, node)
├── examples/        — Usage examples
└── docs/            — Documentation
```

## Installation

Clone the repository

```bash
git clone git@github.com:manikantamedindi/my-ai-toolkit.git
```

Configure OpenCode to load the repository according to your preferred setup (local path or Git-based plugin, depending on the version you use).
