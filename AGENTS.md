# AGENTS.md — my-ai-toolkit

## What this is

A skeleton repo for reusable OpenCode skills, agents, prompt templates, project templates, and workflows. The planned directory structure (from `README.md`) is:

```
skills/        - Reusable AI Skills
agents/        - Specialized coding agents
prompts/       - Prompt templates
templates/     - Project templates
docs/          - Documentation
```

## Current state

- Skeleton structure created with directories for skills/, agents/, prompts/, templates/, examples/, and docs/.
- Agent definitions (.md files) exist in agents/ with brief descriptions.
- Prompt templates (.md files) exist in prompts/ with brief descriptions.
- Documentation stubs exist in docs/.
- No build, test, lint, or typecheck scripts exist.
- No `package.json`, `opencode.json`, or any config files.

## Remote

- `origin` → `git@github.com:manikantamedindi/my-ai-toolkit.git`

## Adding content

Each directory is ready for content. Create skill files under the appropriate `skills/<domain>/` subdirectory. Add new agent definitions as `.md` files in `agents/`. Add prompt templates as `.md` files in `prompts/`. Full project templates go under `templates/`. There are no established conventions yet — establish them as you go.
